About this website

This website uses CSS and HTML only and was written almost entirely in Vim. It's DOCTYPE is HTML5, even though no HTML5 features are used here.
It's purpose is to share all the knowledge I gather in my learning phase in terms of programming in hopes of it being useful to people.
The emphasis here lies on open source and everything associated with it.
In a technical sense, the goal of this website is to keep it small, simple and therefore, make the content load in a fast and reliable manner.
It doesn't use any CGI scripts or JavaScript.
