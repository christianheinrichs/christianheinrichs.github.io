Про этот сайт

Этот сайт использовать только CSS и HTML и был написан почти полностью с программий Vim. В техническом смысле, цель этого сайта, использовать рессурсы, чем меньше содержание тем быстрее и надёжнее способ загрузки. Не использовать CGI-скрипты и JavaScript тоже.
