Über diese Website

Diese Website verwendet nur CSS und HTML und wurde fast gänzlich mit dem Programm Vim geschrieben. Der DOCTYPE ist HTML5, obwohl hier keine HTML5-Features verwendet werden. Im technischen Sinne ist das Ziel dieser Website, die Ressourcen so einfach und klein wie möglich zu halten und somit den Inhalt in einer schnellen und zuverlässigen Weise laden zu lassen. Es werden keine CGI-Skripte und auch kein JavaScript verwendet. Diese Seite wird zur Verfügung gestellt von GitLab Pages.
