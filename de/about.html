Über diese Website

Diese Website verwendet nur CSS und HTML und wurde fast gänzlich mit Vim geschrieben.
DOCTYPE ist HTML5, obwohl hier keine HTML5 Features verwendet werden.
Momentan wird diese Homepage komplett statisch getippt, hoffentlich ändert sich das jedoch.
Der Sinn dieser Website ist es all das Wissen, welches ich in meiner Lernphase im Bezug aufs Programmieren ergattere zu teilen,
in der Hoffnung, dass es einigen Leuten nützt.
Hier liegt die Betonung auf Open-Source und allem was damit assoziiert ist.
Im technischen Sinne ist das Ziel dieser Website, die Ressourcen so einfach und klein wie möglich zu halten und somit den Inhalt in einer schnellen und zuverlässigen Weise laden zu lassen.
Es werden keine keine CGI-Scripts und auch kein JavaScript verwendet.
