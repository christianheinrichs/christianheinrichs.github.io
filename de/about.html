Über diese Website

Diese Website verwendet nur CSS und HTML. Sie wurde fast gänzlich mit Vim geschrieben. Der DOCTYPE ist HTML5, obwohl hier keine HTML5 Features verwendet werden
Der Sinn dieser Website ist es, all das Wissen, welches ich in meiner Lernphase im Bezug aufs Programmieren ergattere zu teilen, in der Hoffnung, dass es einigen Leuten nützt.
Die Betonung hier liegt auf Open-Source und allem was damit assoziiert ist.
Das Ziel dieser Website ist es, alles einfach zu halten und so den Inhalt in einer schnellen und zuverlässigen Weise laden zu lassen.
Sie benutzt keine CGI-Scripts oder JavaScript.
